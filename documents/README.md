## 🛠️ 开发工具

### 搜索命令

```bash
# 在项目中搜索常量
grep -r "CONSTANT_NAME" --include="*.php"

# 搜索函数定义
grep -r "^function function_name" --include="*.php"

# 搜索配置使用
grep -r "C\('config_key'\)" --include="*.php"

# 搜索Session变量
grep -r "\$_SESSION\['key'\]" --include="*.php"
```
